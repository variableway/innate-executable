# 教程编写完整示例

> 本教程演示 Innate Desktop 教程系统中支持的所有 Markdown 语法和自定义 MDX 组件。你可以直接复制本文件作为编写新教程的起点。

---

## 1. 基础 Markdown 语法

### 1.1 文本格式

你可以使用 **粗体**、*斜体*、`行内代码` 来强调内容。

这是一个 [外部链接](https://example.com)，会在新标签页打开。

### 1.2 列表

无序列表：

- 第一项
- 第二项
  - 嵌套子项 A
  - 嵌套子项 B
- 第三项

有序列表：

1. 步骤一
2. 步骤二
3. 步骤三

任务列表（GFM）：

- [x] 已完成的事项
- [ ] 待办事项一
- [ ] 待办事项二

### 1.3 引用块

> 这是一个引用块。可以用来强调重要提示、名言或补充说明。
>
> 支持多行内容。

### 1.4 表格

| 功能 | macOS/Linux | Windows | 说明 |
|------|-------------|---------|------|
| 安装 | `curl \| bash` | `irm \| iex` | 一键安装 |
| 更新 | `claw update` | `claw update` | 自动更新 |
| 卸载 | `claw uninstall` | `claw uninstall` | 清理配置 |

## 2. 代码块

### 2.1 普通代码块（自动高亮）

普通 fenced code block 会自动使用 **Shiki** 进行语法高亮：

```python
# Python 示例
import random

def greet(name: str) -> str:
    return f"Hello, {name}!"

names = ["Alice", "Bob", "Charlie"]
print(greet(random.choice(names)))
```

```javascript
// JavaScript 示例
const config = {
  model: "glm-5.1",
  temperature: 0.7,
  skills: ["search", "email", "calendar"],
};

async function initClaw() {
  const claw = await OpenClaw.create(config);
  return claw;
}
```

```json
{
  "name": "my-project",
  "version": "1.0.0",
  "dependencies": {
    "react": "^19.0.0",
    "next": "^16.0.0"
  }
}
```


## 3. MDX 自定义组件

### 3.1 RunnableCodeBlock — 可运行的代码块

`<RunnableCodeBlock>` 会渲染一个带 **复制** 和 **运行** 按钮的代码块。点击「运行」会将代码发送到内置终端执行。

单行命令：

<RunnableCodeBlock code="node --version" language="bash" />

多行脚本：

<RunnableCodeBlock
  code="npm install -g typescript && tsc --init"
  language="bash"
/>

设置为不可运行：

<RunnableCodeBlock
  code="this is just a demo, not runnable"
  language="text"
  runnable={false}
/>

### 3.2 PlatformTabs — 平台切换

当同一条命令在 macOS/Linux 和 Windows 上有不同写法时，使用 `<PlatformTabs>`：

<PlatformTabs
  unix={
    <RunnableCodeBlock
      code="curl -fsSL https://example.com/install.sh | bash"
      language="bash"
    />
  }
  windows={
    <RunnableCodeBlock
      code="irm https://example.com/install.ps1 | iex"
      language="powershell"
    />
  }
/>

默认选中 Windows：

<PlatformTabs
  unix={
    <RunnableCodeBlock code="export API_KEY=xxx" language="bash" />
  }
  windows={
    <RunnableCodeBlock code="$env:API_KEY = 'xxx'" language="powershell" />
  }
  defaultPlatform="windows"
/>

### 3.3 RunButton — 单行命令按钮

`<RunButton>` 渲染一个醒目的按钮，适合在教学步骤中引导用户执行某个命令：

先检查安装状态：

<RunButton command="claw --version" />

然后启动服务：

<RunButton command="claw start" />

## 5. 高级技巧

### 5.1 在代码块中包含 JSX

如果需要在代码块中展示 JSX 代码（例如教程本身在教用户写 MDX），直接写 fenced code block 即可，不需要特殊处理：

```mdx
---
title: "我的教程"
---

# 欢迎使用

<PlatformTabs
  unix={<RunnableCodeBlock code="npm install" language="bash" />}
  windows={<RunnableCodeBlock code="npm install" language="powershell" />}
/>
```

### 5.2 混合使用组件和 Markdown

你可以在一段说明文字中穿插组件：

首先，确保你已经安装了 Node.js：

<RunnableCodeBlock code="node --version" language="bash" />

如果看到版本号输出，说明安装成功。接下来安装项目依赖：

<RunnableCodeBlock code="npm install" language="bash" />

安装完成后，启动开发服务器：

<RunButton command="npm run dev" />
